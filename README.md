## Grade Calculator – Java Mini Project
The Grade Calculator is a Java-based mini project that calculates a student’s total marks, percentage, and final grade based on marks entered for multiple subjects. It helps automate result calculation and reduces manual errors.

## Technologies Used
Java
Java Scanner class

## How to Run
  1.Open the project in any Java IDE (Eclipse / IntelliJ / VS Code)
  2.Compile the program
  3.Run the main class
  4.Enter subject marks when prompted

## Grade Criteria (Example)
  90% & above – A
  75% – 89% – B
  50% – 74% – C
  Below 50% – F
